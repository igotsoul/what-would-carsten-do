# "Faces & Emotions" with Carsten Lützen

## Summary
Carsten Lützen shares a simple, systematic method for drawing expressive faces using only straight and curved lines. By combining just two eye shapes with four mouth shapes in a matrix, facilitators can quickly produce a wide range of recognizable emotions for use in visual facilitation work — no artistic skill required.

## Methods & Techniques

### Faces & Emotions Combination Matrix (ab 00:45)
- **What:** A matrix of eye-and-mouth combinations using only straight or curved lines to generate varied facial expressions. Every combination produces a valid, readable face.
- **When to use:** When creating visual facilitation materials, sketching figures on flipcharts or whiteboards, or building a personal drawing vocabulary for facilitation contexts.
- **Group size:** N/A — this is a personal drawing practice technique.
- **Duration:** Can be practiced in minutes; the matrix itself takes under 5 minutes to draw.
- **How it works:**
  1. Choose one of two eye shapes (e.g., neutral dots or angled lines).
  2. Choose one of four mouth shapes — a curved line up (smile), curved line down (frown), straight line (neutral/indifferent), or wavy/angled line (questioning).
  3. Combine eyes and mouth. Every combination yields a valid, recognizable emotion.
  4. Carsten identifies emotions such as: happy, sad, embarrassed, furious, questioning, resigned, indifferent, evil/scheming, and anxious — all from the same small set of lines.
- **Source:** https://www.youtube.com/watch?v=RKaCYr5qM64&t=45s

### Line Offset & Positioning (ab 04:00)
- **What:** Shifting where elements are placed on the face (e.g., moving the mouth off-center) changes the emotional read significantly without adding complexity.
- **When to use:** When a standard combination doesn't quite capture the intended emotion — use placement adjustments to fine-tune expression.
- **How it works:** Draw standard eyes, then deliberately offset the mouth position left, right, or lower/higher. Vary line length slightly. Try it, then put words to it and see if it matches the intended emotion.
- **Source:** https://www.youtube.com/watch?v=RKaCYr5qM64&t=240s

## Key Principles
- Simplicity over artistry: the goal is a reproducible visual vocabulary, not polished illustration. Simple shapes that reliably communicate emotion are more valuable than detailed drawings.
- Constraints generate creativity: limiting yourself to straight and curved lines forces a systematic approach that actually yields more variety than freeform drawing.
- Every combination is valid: there are no wrong faces — any pairing of eyes and mouth produces a face worth interpreting. This removes the fear of drawing incorrectly.
- Words bridge the gap: if a face doesn't immediately read as the target emotion, labeling it resolves any ambiguity. Drawing and language work together.

## Facilitation Tips
- Carsten recommends building a personal reference matrix — draw it once, keep it, and use it as a cheat sheet during live facilitation sessions.
- Don't overthink individual expressions. Draw, observe what emotion it suggests, and label it. Let the drawing lead.
- Practice the matrix to the point where the combinations become automatic — this frees cognitive load during actual facilitation.
- Invite others to interpret the same face differently; multiple valid readings can spark useful conversation about emotion and perception.

## Best Used When
- Preparing flipcharts, templates, or facilitation materials that include human figures with emotional states.
- Building a visual facilitation drawing practice from scratch, especially for non-artists.
- Wanting to add emotional nuance to simple stick figures without increasing drawing complexity.
